# Customer Behaviour Analysis Ecommerce

## Description
This project analyzes customer behavior for an e-commerce business. It includes data cleaning, exploratory analysis, SQL-based data handling, and visualization assets to help identify patterns in customer shopping habits.

## Tools
- Python
- Jupyter Notebook
- SQL
- Power BI
- Git

## Abstracts

- SQL analysis: SQL scripts are used to query data, generate insights, and support reporting.
- Visualization: Visual assets are created to highlight trends, customer segments, and purchase behavior.
